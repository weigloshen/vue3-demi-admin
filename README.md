## 简介

`DemiAdmin`是一个新手学生写的前端中后台模板，使用了 `vue3`, `vite4`, `pinia`, `element plus`, `typescript`, `unocss`, `plop`

## 安装查看

- 获取代码

```js
https://github.com/weigloshen/vue3-demi-admin.git
```
- 安装依赖

```js
cd vue3-demi-admin && pnpm install
```

- 运行
```js
pnpm dev
```

- 打包
```js
pnpm build
```
